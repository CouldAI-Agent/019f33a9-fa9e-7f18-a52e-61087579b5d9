# Dipongkar Barmon - Premium Portfolio

A premium, modern, and highly animated personal portfolio built for Dipongkar Barmon. It showcases skills in Full Stack Development, Machine Learning, and Software Engineering.

## Features
- **Hero Section**: Animated introduction with profile and quick links.
- **About Me**: Professional summary and background.
- **Skills**: Categorized skills in Programming, Frontend, Backend, Database, and ML.
- **Responsive Layout**: Designed to work smoothly on mobile, tablet, desktop, and web.
- **Premium Aesthetics**: Uses elegant typography, minimal UI, soft shadows, and clean styling.

## Tech Stack
- Flutter (Dart)
- Cross-platform support (Web, iOS, Android, Desktop)

## Setup & Run
1. Ensure you have Flutter installed.
2. Run `flutter pub get` to install dependencies.
3. Run `flutter run -d chrome` to test on the web, or run on your preferred device.
